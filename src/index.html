@apply to customise css

use @layer to change default classes

For example, using @layer base {} to change the styling of the anchor tag will override the base styles in the given utility classes

This will mean that, for example, you changed the anchor styling. ALL of the anchor tags will apply your new styling

Component Directives

When you will be repeaating something over and over again, it may be useful to put all of the utility classes into a component.

Some examples of components that might be useful:

  • buttons
  • dropdowns
  • logins
  • modals
  • tabs
  • inputs
  • selects

Example - button

@layer components { .btn-blue { @apply py-2 px-4 bg-blue-500 text-white text-center uppercase rounded-md hover:bg-blue-300 hover:text-blue-900; }
